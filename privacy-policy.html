隐私政策

最后更新日期：2026 年 5 月 21 日

欢迎使用我们的 Chrome 扩展（以下简称"本扩展"或"本产品"）。本隐私政策旨在说明我们如何收集、使用和保护您的个人信息。本政策适用于我们发布的所有 Chrome 扩展产品。

一、信息收集

本扩展在正常运行过程中可能会收集以下信息：

  • 网页内容数据：根据扩展的具体功能，我们可能会读取当前页面的公开数据（如页面文字、元素内容等），用于在本地进行处理和分析。
  • 存储数据：我们会将您的使用偏好设置（如功能配置、筛选条件等）存储在浏览器的本地存储（chrome.storage）中。
  • API 请求数据：部分扩展可能会与第三方网站的公开 API 进行交互，以获取实现功能所必需的数据。

二、信息使用

我们收集的信息仅用于以下目的：

  • 实现扩展的核心功能
  • 保存您的个性化设置
  • 改善用户体验和扩展功能

三、信息共享

我们不会收集、存储或向任何第三方传输您的个人信息。所有数据处理均在您的浏览器本地完成，不会上传到任何服务器。

四、数据安全

由于所有数据均在本地处理，不涉及网络传输，您的数据安全由浏览器沙箱环境提供保护。但我们仍建议您保持浏览器和扩展为最新版本。

五、第三方服务

本扩展可能会与第三方网站或服务的公开 API 进行交互，以实现所需功能。这些交互遵循相关服务条款，且不涉及您的个人身份信息。

六、政策更新

我们可能会不时更新本隐私政策。重大变更时，我们会在扩展的更新说明中通知您。建议您定期查阅本页面以了解最新信息。
